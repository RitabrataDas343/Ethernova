## Starting the Project

### Requirements:

Please follow the guides of the following before starting to work on the project:
* [NodeJS](https://nodejs.org/en/download/)
* [npm](https://docs.npmjs.com/cli/install)

1. Clone the Repository

   ```bash
   git clone https://github.com/RitabrataDas343/Ethernova.git
   cd Ethernova
   ```
  
2. Install the dependencies

   ```bash
   sudo apt install npm
   npm install
   sudo npm install truffle
   ```
   
3. Configure truffle 

   ```bash
   truffle develop
   compile
   migrate
   ```
   
5. Run the project

   ```bash
   npm start
   ``` 
   
4. Head to server http://localhost:4200
